Warning !

You are attempting to run PSFj using 32-bit version of Java. This limits the available amount of memory to %max_memory% MB of memory even if your system has more memory available. In case large bead image stacks are used, this can cause a crash of the software.

To use PSFj with more memory capacity, please run PSFj on a 64-bit OS with a 64-bit Java Runtime installed.

So, if your OS is 64-bit, please reinstall a 64-bit version of Java.
